1) Create a virtual environment in Ubuntu

	virtualenv -p python3.10 $HOME/tmp/masterseminar/

2) Activating the environment

	source $HOME/tmp/masterseminar/bin/activate
	
3) Install required packages

    • pip install PyAudio
    • pip install pyttsx3
    • pip install SpeechRecognition
    • pip install geocoder
      
4) Run the python file
      python sensation.py

5) Say the commands like "Start sensation" "What is the time" "Stop sensation" to test the code
